# SaaS Churn Analysis (SQL)

**Author:** Anuj Saini  
**Description:** An end-to-end SQL-based analysis of customer churn metrics and retention patterns for a subscription-based SaaS platform.

---

## Project Structure

This repository is structured as follows:

*   **`sql/`**: Contains raw and optimized SQL queries used to extract and analyze SaaS churn and subscriber metrics.
*   **`findings/`**: Detailed analysis reports, visualization summaries, and strategic recommendations.
*   **`notebooks/`**: Jupyter notebooks for exploratory data analysis (EDA) and data visualization.

---

## Focus Areas

1.  **Churn Rate Trends**: Analyzing monthly, quarterly, and annual subscriber churn.
2.  **Cohort Analysis**: Tracking user retention rates over cohorts grouped by sign-up month.
3.  **Customer Lifetime Value (LTV)**: Evaluating customer segment LTV against acquisition costs.
4.  **Revenue Churn**: Distinguishing between customer churn and monthly recurring revenue (MRR) churn.
